# 附录 F：GitHub Sponsors 开源赞助指南 💰

> 让开源创作可持续——了解如何通过 GitHub Sponsors 赞助他人或接收赞助。

## 什么是 GitHub Sponsors？

**GitHub Sponsors** 是 GitHub 官方提供的开源赞助系统，让开发者可以直接通过 GitHub 赞助/接收赞助，支持开源项目的可持续发展。

### 核心理念

```
开发者投入时间维护开源项目  →  GitHub Sponsors 平台  →  用户/企业赞助支持
        ↓                           ↓                           ↓
  获得经济回报              GitHub 作为可信中间人          获得高质量开源软件
```

## 配置 Sponsors 个人页（面向维护者）

### 开通条件

- ✅ 拥有 GitHub 账号
- ✅ 有活跃的开源项目贡献记录
- ✅ 绑定 Stripe 账户（用于收款）
- ✅ 所在地区支持 GitHub Sponsors

### 开通步骤

```
第1步：个人主页 → 点击 "Sponsors" 标签
                        ↓
第2步：点击 "Set up sponsorship profile"
                        ↓
第3步：绑定 Stripe 账户（收款用）
                        ↓
第4步：设置赞助层级和权益描述
                        ↓
第5步：审核通过 → Sponsors 页面上线 🎉
```

### 配置 FUNDING.yml

在仓库根目录创建 `.github/FUNDING.yml`，配置赞助链接：

```yaml
# 方式1：GitHub Sponsors
github: [GTX950L]     # 你的 GitHub 用户名

# 方式2：自定义链接
custom: ["https://你的网站", "https://afdian.com/你的页面"]
```

配置后，仓库顶部会显示 "Sponsor" 按钮：

```
┌────────────────────────────────────────────┐
│ ⭐ Star  🍴 Fork  💖 Sponsor  🔔 Watch    │
└────────────────────────────────────────────┘
```

## 如何赞助开源项目（面向赞助者）

### 找到赞助入口

```
方法1：直接在项目 README 中查看 Sponsor 按钮
方法2：在仓库主页右上角找 💖 Sponsor 按钮
方法3：访问作者个人主页 → Sponsors 标签
```

### 赞助方式

| 方式 | 说明 | 特点 |
|------|------|------|
| **一次性赞助** | 单次支付 | 灵活、无承诺 |
| **月度赞助** | 每月自动扣款 | 持续支持、可随时取消 |
| **企业赞助** | 企业级支付 | 可开发票、含商业权益 |

### 查看和管理赞助

登录 GitHub → 个人 Settings → **Sponsors** 标签页

## 最佳实践

### 维护者角度

```
✅ 推荐做法：
├── 清晰描述每个层级的权益
├── 在 README 中用 Sponsor 徽章
├── 定期感谢赞助者（在 Release Notes 中）
├── 维护者之间互相 Sponsorship
└── 透明公开资金用途

❌ 避免做法：
├── 将赞助与代码合并挂钩（不道德）
├── 过度宣传赞助链接
└── 区别对待赞助者和非赞助者
```

### 赞助者角度

```
✅ 推荐做法：
├── 优先赞助你日常使用的开源项目
├── 考虑月度赞助而非一次性
├── 企业用户可批量赞助依赖项目
└── 签署开源协议时考虑资助条款

❌ 避免做法：
├── 用赞助换取特殊待遇
└── 赞助后期望项目按你的需求开发
```

## 常见问题

**Q: 我人在中国，可以开通 GitHub Sponsors 吗？**  
A: ✅ 可以。GitHub Sponsors 支持 Stripe 支付，只要有 Stripe 支持的银行卡即可收款。

**Q: 赞助金额 GitHub 抽成吗？**  
A: 目前 GitHub Sponsors **不收取平台费用**，100% 赞助金额归开发者（仅 Stripe 收取少量手续费）。

**Q: 我赞助后能开发票吗？**  
A: ✅ 企业赞助可申请发票（需联系 GitHub 支持）。

**Q: 除了 GitHub Sponsors，还有哪些开源赞助平台？**  
A: 常见的还有 Patreon、Open Collective、爱发电（国内）等。
